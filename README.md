# 购物车+信用卡管理（作业代码）
 bin/atm.py 程序入口文件  
 core/login.py 登录模块  
 core/shop.py 购物模块  
 core/xinyonka.py 信用卡模块  
 core/main.py 主程序  
 db/user.txt 用户数据  
 db/op_log.txt  操作日志  
 db/credit_card.txt 信用卡数据  
 db/shopping_car/  用户购物车
功能：  
1.用户注册，提交信用卡卡号，初始额度10000  
2.用户登录，进入商城，购买商品，扣除额度  
3.对信用卡进行查询，还款，取款功能  
4.将用户操作写入日志op_log.txt
